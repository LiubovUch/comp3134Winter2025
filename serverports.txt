1. A server port is a communication endpoint used by the operating system to distinguish network traffic for different services or applications on the same device.

2. The server port range spans from 0 to 65535.

3. 16 bits.

4. 10 Common Server Ports:

	20 - FTP 
	21 - FTP 
	22 - SSH 
	23 - Telnet
	25 - SMTP 
	53 - DNS 
	80 - HTTP 
	110 - POP3 
	143 - IMAP 
	443 - HTTPS
